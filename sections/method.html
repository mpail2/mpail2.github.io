The Algorithm - MPAIL2 We introduce Model Predictive Adversarial Imitation Learning 2 (MPAIL2), an IRL algorithm that learns a world model and performs latent planning.


1. Five (5) components comprise MPAIL2's world model: encoder, dynamics, reward, value, and policy. Online, a planner uses these components to sample plans, predict their outcomes, evaluate them, and finally select a plan.

2. The learner's task is given through observations of task demonstrations, like watching a person cook. They are used only when learning the reward.
