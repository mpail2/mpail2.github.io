Transfer Learning Robot is trained using an initial set of demonstrations for one task. Then, the model weights are directly transferred for training on a new set of demonstrations for a different task. Since the tasks are similar, we expect the robot to learn new skills quickly by relying on its continual experience. However, this has not been true of most RL methods without specialized methods or massive data coverage.
